SCAMCEA é uma metodologia derivada do SCAMPER, framework criativo desenvolvido por Bob Eberle 1971 baseado nos trabalhos de Alex Osborn criador do Brainstorming. O SCAMCEA adapta o modelo original para contexto brasileiro de PMEs, mantendo 7 diretrizes criativas para gerar inovação sistemática TITLE SCAMCEA - Framework de Inovação para Produtos e Serviços - ORIGEM E CONCEITO

## Como Preencher

1. **CONDIÇÃO ATUAL** Descreva a situação/característica atual do produto materiais, processos, forma, usos, etc.
2. **OPÇÕES A/B/C** Para cada diretriz SCAMCEA, gere 3 ideias criativas de como aplicá-la
3. **CUSTO** Estimativa de investimento necessário baixo/médio/alto ou valores
4. **BENEFÍCIO** Ganho esperado economia, aumento vendas, diferenciação, etc.
5. **IMPLEMENTAÇÃO** Classifique viabilidade 1=fácil, 2=moderado, 3=difícil
6. **OBSERVAÇÕES** Riscos, dependências ou bloqueadores identificados

## QUANDO USAR SCAMCEA

Ideal para:
- Inovação incremental em produtos/serviços existentes
- Brainstorming estruturado de melhorias
- Diferenciação competitiva em mercados saturados
- Redução de custos via simplificação/eliminação
- Expansão de mercado novos usos/públicos

Não recomendado para:
- Validação de demanda use Pesquisa de Mercado
- Definição de modelo de negócio use Business Model Canvas
- Planejamento estratégico completo use SWOT + PDCA

## INTEGRAÇÃO COM BUSINESS MODEL CANVAS

O SCAMCEA complementa o Canvas respondendo:
- **Proposta de Valor** Como torná-la única? Modificar, Combinar
- **Atividades-Chave** O que eliminar? O que substituir? Eliminar, Substituir
- **Recursos-Chave** Como otimizar? Arranjar, Adaptar
- **Canais** Novos usos/públicos? Colocar Outros Usos
- **Estrutura de Custos** Onde reduzir? Eliminar, Substituir

Fluxo recomendado:
1. Preencher Business Model Canvas visão macro do negócio
2. Aplicar SCAMCEA em cada bloco do Canvas inovação tática
3. Priorizar ideias usando Matriz GUT ou ROI
4. Implementar mudanças e medir resultados PDCA
